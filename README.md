# DeepPDF-Summarizer

This tool allows users to extract text from PDF files using PyMuPDF and OCR (via Tesseract). It also provides functionality to summarize PDF content using the DeepSeek API.

## Requirements

- Python 3.x
- PyMuPDF: `pip install PyMuPDF`
- Tesseract OCR: `pip install pytesseract`
- Pillow: `pip install Pillow`
- DeepSeek API (you need to provide an API key)

## How to Use

1. Clone this repository:
    ```bash
    git clone https://github.com/phoenix060505/DeepPDF-Summarizer.git
    ```
2. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3. Run the application and provide your DeepSeek API key, PDF folder path, and custom instruction.
